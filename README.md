# Hostel_MIS
This is a web-based hostel MIS developed using MySQL and PHP for the back end and HTML/CSS for the front end
Hostels Management System Using PHP and MySQL
Installation Steps(Configuration)
1. Download and Unzip the file on your local system.
2. Put this file inside xampp/htdocs/ .
3. Database Configuration
Open phpmyadmin
Create Database hostels.
Import database hostels.sql
Open Your browser put inside browser “http://localhost/hostels/”
